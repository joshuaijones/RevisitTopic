Buttons

Styling buttons are an important way of guiding users. Buttons styling should be easily readable and improve the user experience (UX). Most styling can be done in a CSS file.

Below are examples of some of the properties that may increase UX: 

                         
.btn {
    background-color: #2176FF;           // assigning contrasted background colors
    border-radius: 30px;                 // assign a border radius to appear less blocky
    color: #ffffff;                      // constrast color of text with background
    font-family: 'Raleway',sans-serif;   // assign a font family that is appropriate
    font-size: 16px;                     // large enough text size
    font-weight: 400;                    // font strength may be useful to change
    padding: 14px;                       // padding increases readability
    text-decoration: none;               // remove or keep text-underline
    text-transform: uppercase;           // may change text to be uppercase   
    cursor: pointer;                     // communicate with pointer
  }
  
  .btn:hover {                           // change when the cursor hovers over 
    background-color: lightblue;         // change background
    color: darkblue;                     // change color
    transition: color .4s ease-in,       // add a smooth transition
    background-color .4s ease-in;
  }
  
  .btn:active {                          // change when button is pressed
    font-size: 18px;                     // change font size for readability
    transition: font-size .1s ease-in-out;
  }

Tables

Tables are a useful way of displaying information to a user. Displaying data in through HTML tables utilizes a row and column system to organize data.

The standard HTML table format utilizes a basic structure: 

                    
<table>
  <tr>
    <th>item1</th>
    <th>item2</th>
    <th>item3</th>
  </tr>
  <tr>
    <td>type1</td>
    <td>type2</td>
    <td>type3</td>
  </tr>
  <tr>
    <td>kind1</td>
    <td>kind2</td>
    <td>kind3</td>
  </tr>
</table>

Table tags indicate where the table starts and ends. The tags <tr> reference the start and end of a table row. Table items occupy the same row in a line. The tag <th> refer to table headings while the tag <td> refer to table data.

Tables may use semantic dividers to organize code. These code tags include: <thead> , <tbody> , and <tfoot>. These should be utilized similar to the use of <header> , <main> , and <footer> for tables.
